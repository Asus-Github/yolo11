#### 2.3 主干网络融合 Triplet Attention（约 1000 字 + 公式 5-7 个 + 1 图）

**仿范文 2.3 节"主干网络融合注意力机制"**：

##### 段 1：与传统注意力对比
- SE（仅通道）、CBAM（串行通道+空间）、ECA（轻量通道）的不足
- **Triplet Attention 优势**：三支并行、跨维度交互、参数量极少

##### 段 2：原理与公式
**公式 (1)** Z-Pool 操作（最大值与平均值拼接）：
```
Z-Pool(X) = Concat(MaxPool(X), AvgPool(X))
```

**公式 (2)** 通道-高度分支（C×H 维度交互）：
```
A_ch = σ(Conv₇ₓ₇(Z-Pool_w(X)))
```

**公式 (3)** 通道-宽度分支（C×W 维度交互）：
```
A_cw = σ(Conv₇ₓ₇(Z-Pool_h(X)))
```

**公式 (4)** 空间分支（H×W）：
```
A_sp = σ(Conv₇ₓ₇(Z-Pool_c(X)))
```

**公式 (5)** 三支并行融合：
```
Y = (1/3) · (X̃_ch + X̃_cw + X_sp)
其中 X̃_ch、X̃_cw 为对应分支转置-加权-反转置后的特征
```

**公式 (6)** 嵌入 YOLOv11 的残差连接（仿范文式 7）：
```
X_out = BN(TripletAttn(X_in)) + X_in
```

#### 2.4 动态检测头 DyHead（约 1000 字 + 公式 4-5 个 + 1 图）

**仿范文 2.4 节"简化的双向多尺度特征融合"写法**：

##### 段 1：原检测头的问题
- YOLOv11 原解耦头的多尺度特征处理是"独立计算"，无跨层级动态交互。

##### 段 2：DyHead 三类注意力
**公式 (7)** 尺度感知注意力（关注 L 维度）：
```
π_L(F) · F = σ(f(αₗ(F))) · F
其中 αₗ 为 L 维度全局平均池化
```

**公式 (8)** 空间感知注意力（关注 SC 维度，使用可变形卷积）：
```
π_S(F) · F = (1/L) Σ_{l=1}^L Σ_{k=1}^K w_{l,k} · F(l, p_k+Δp_k, c) · Δm_k
```

**公式 (9)** 任务感知注意力（关注 C 维度）：
```
π_C(F) = max(α¹·F + β¹, α²·F + β²)
其中 α, β 为任务相关的可学习参数
```

**公式 (10)** DyHead block 级联：
```
W(F) = π_C(π_S(π_L(F) · F) · F) · F
```

##### 段 3：堆叠与嵌入
- DyHead 通常堆叠 6 个 block（论文需说明你用几个）
- 替换原 Detect Head 的位置
- **图 5**：DyHead 结构图（三类注意力级联）

#### 2.5 Wise-IoU 损失函数（约 800 字 + 公式 4 个）

**完全对应范文 2.6 节，可直接借鉴写法**：

##### 段 1：CIoU 的不足
- 几何因素惩罚过重，对低质量样本"过度惩罚"导致收敛不稳定。
- 复合交叉面积 + 修正因子计算复杂，增加训练负担。

##### 段 2：Wise-IoU 三层演进

**公式 (11)** 基础 IoU 损失：
```
L_IoU = 1 - IoU
```

**公式 (12)** WIoU v1 距离注意力：
```
L_WIoU_v1 = R_WIoU · L_IoU
R_WIoU = exp[((x − x_gt)² + (y − y_gt)²) / (W_g² + H_g²)*]
其中 * 表示从计算图分离梯度（停止反向传播）
```

**公式 (13)** 离群度 β 定义：
```
β = L_IoU* / L_IoU_mean ∈ [0, +∞)
```

**公式 (14)** WIoU v3 动态非单调聚焦因子：
```
r = β / (δ · α^(β-δ))
L_WIoU_v3 = r · R_WIoU · L_IoU
其中 α、δ 为超参数（α=1.9, δ=3）
```
